# Note-Taking

## Zettelkasten

- Best for researchers, non-fiction authors, and building expertise over years rather than weeks.
- Goal is not to create personal knowledge management. The goal is to use the main cards to create output that you will share with others.
- Four sections
  - Index cards.
    - Essentially table of contents.
    - Use alphabetical index cards as dividers.
    - Use it to store keywords, and other information to easily retrieve main cards.
  - Main cards.
    - Convert bibliography card to main cards.
    - Reference [Source (Book) Reading](#source-book-reading) section.
  - Bibliography cards.
    - Use when reading a source.
    - Reference [Source (Book) Reading](#source-book-reading) section.
  - Catalog.
    - Store list of all sources.
    - Number all the sources starting from one.
    - Reference the source number in other sections, to know where the information/idea came from.

### Source (Book) Reading

Create bibliography card.

- On index card write name of source, and source number.
- Create left-margin to write page number.
- Write short description about the information in the page.
- In the book, use square brackets to identify where the short description is pointing to.
- After reading the book, wait 2-4 weeks before writing main cards. Time helps filter out the unnecessary notes.

Create main cards.

- Use the source material to create categories.
  - Use an index card and write category name to act as divider.
  - Assign the category a number starting from 1 and write it in top-right corner.
  - Create subcategoires and assign it a number like 1/1.
- On main card.
  - Write card number in top-right. If the category is 1/1, then the first card number is 1/1/1.
    - Find the logically correct place for it. If that place is between 1/2 and 1/3, then number the card 1/2.5.
  - Put source number and the page number from the source at the bottom.
  - Write title of card at top, and then the main body of the card.
- Branching. Use when you want to dive deeper into a topic.
  - If main card has number 1/2, use 1/2-1 for the branching card.

Create index card.

- For each category, write the category name is the alphabetical card and the category number.
